## Tips for getting PRs merged

1. **Read the contributing guide** of each list before opening the PR — some require a specific format (trailing period, no bold, specific section order).
2. **Check alphabetical order** — most awesome lists sort entries A-Z within sections. Place the entry correctly.
3. **One PR per list** — don't batch multiple additions in one PR.
4. **Be patient** — awesome list maintainers are volunteers. Follow up after 2 weeks if no response.
5. **Star the list repo first** — it's good etiquette and increases the chance of a positive review.
